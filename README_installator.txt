Instalacja zakonczona. Jesli jeszcze tego nie zrobiles, wykonaj ponizsze czynnosci przed przejsciem dalej:

1.Zainstaluj Pythona 3.7 (https://www.python.org/ftp/python/3.7.0/python-3.7.0.exe).
  1.1.Podczas instalacji zaznacz Install launcher for all users (recommended) i Add Python 3.7 to PATH.
  
2.Zainstaluj Gita (https://github.com/git-for-windows/git/releases/download/v2.20.1.windows.1/Git-2.20.1-64-bit.exe).
  2.1.Podczas instalacji zaznacz Use Git from the Windows Command Prompt, Checkout Windows-style, commit Unix-style endings, i Use MinTTY (the default terminal MSYS2).

3.Uruchom program updater.exe, znajdujacy sie w folderze glownym aplikacji.

Po wykonaniu tych czynnosci mozesz bez przeszkod uruchomic bota (plik setup.exe w folderze glownym).
Jesli chcesz uzyskac pomoc lub wiecej informacji odszukaj plik README.md w folderze glownym aplikacji.
